# TravelMate AI v18.0

## 주요 변경사항
- 일정 카드마다 **📎 첨부·메모** 버튼 표시
- 일정별 사진, PDF, 문서 첨부/열기/삭제
- 예약번호, 연락처, 홈페이지 링크, 메모 저장
- IndexedDB 저장으로 앱 업데이트 후에도 데이터 유지
- GitHub Pages 새 배포 자동 감지 및 서비스워커 캐시 자동 교체
- 음성 인식 안정화 코드 유지

## 사용 방법
1. ZIP 압축을 풉니다.
2. GitHub `Hawaii-2026` 저장소의 기존 파일에 모두 덮어씁니다.
3. Commit 후 약 1~2분 뒤 기존 URL을 엽니다.
4. 일정 화면에서 각 일정의 `📎 첨부·메모` 버튼을 누릅니다.

## 첨부파일 저장 방식
첨부파일은 현재 기기의 브라우저/PWA IndexedDB에 저장됩니다. 앱 업데이트에는 유지되지만 다른 기기와 자동 동기화되지는 않습니다. 파일당 최대 20MB입니다.
