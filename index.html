또는 이메일

PC와 휴대폰에서 같은 계정으로 로그인하면 자동 동기화됩니다.

로그인 계정-
마지막 동기화아직 없음

여행 추가·수정·삭제와 일정 변경 시 약 1초 후 자동 저장됩니다.

로그인 전에는 이 기기에만 저장됩니다.

동기화 원칙

1각 기기에서 같은 이메일로 로그인
2앱을 열면 자동으로 데이터 병합
3변경 내용은 자동 저장·자동 반영
