# v15.2 시간 및 남은시간 수정

## 변경 사항
- 하와이 현지 시각과 한국 현재 시각을 동시에 표시
- 일정 날짜와 시간을 모두 포함해 남은 시간 계산
- 여행 전: `48일 12시간 15분 남음`
- 30분 이하: 주황색 강조
- 5분 이하: 빨간색 강조
- 지난 일정은 건너뛰고 다음 일정 자동 선택
- 모든 일정 종료 시 `전체 일정 완료` 표시

## 계산 기준
하와이 일정은 `Pacific/Honolulu` 기준으로 계산합니다.
한국 시간은 `Asia/Seoul` 기준으로 별도 표시합니다.
