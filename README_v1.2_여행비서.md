# TravelMate AI v1.2

## 구현 기능
- 메뉴판 번역: 카메라 촬영과 저장된 사진 불러오기 버튼 분리
- Open-Meteo 기반 현재 여행지 날씨
- 현재 위치에서 다음 일정/직접 입력 목적지 길찾기
- 탑승권·호텔 바우처 이미지/PDF 보관함
- 앱 실행 중 30분·10분 전 브라우저 알림
- AI 여행 비서
- 오프라인 상태 안내 및 기존 일정 로컬 열람
- 기존 자동 동기화, Google/Apple 로그인, 환율, 국가 52개 유지

## 중요한 제한
- 브라우저 알림은 앱이 실행 중일 때 동작합니다. 완전한 백그라운드 푸시는 별도 Push 서버가 필요합니다.
- 바우처 파일은 해당 기기의 브라우저 IndexedDB에 저장되며 클라우드 동기화 대상은 아닙니다.
- AI 여행 비서는 Edge Function이 `mode: "assistant"` 요청을 처리해야 합니다. 기존 함수가 번역/vision만 지원하면 함수 코드 보완이 필요합니다.
